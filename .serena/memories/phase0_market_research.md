# Phase 0 市場調査結果

## 成功事例パターン

### シンプルユーティリティ系の成功例
1. **ポモドーロタイマー**: 月間1200万訪問、広告80% + サブスク20%
2. **電卓アプリ**: 161万DL、累計1180万円（広告1100万 + IAP80万）
3. **禁欲スカウトウォーカー**: 全画面広告で月収2倍（4500円）
4. **暗記メーカー**: 40万DL、5年でアルバイト代替レベル
5. **タスク管理アプリ**: 付箋型TODO、個人開発で生活可能レベル

## 収益化戦略
- **広告メイン**: AdMob（バナー・インタースティシャル・リワード）
- **サブスク補助**: 広告削除、プレミアム機能
- **シンプル実装**: 数時間で広告実装可能

## 避けるべき要素（既存プロジェクトで使用済み）
- 金融計算系（MoneyCounter）
- 習慣トラッカー（Habit Flow）
- 買い物リスト（Quick Shop）
- 植物管理（PlantCare）

## 推奨カテゴリ
- 時間管理・タイマー系
- 学習支援・暗記系
- ライフログ・記録系
- シンプル計算・変換ツール系
