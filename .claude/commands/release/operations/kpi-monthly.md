# 月次KPIレビュー

全アプリの月次KPIを取得してレポートを作成してください。

## データ取得

`WebFetch` で以下のAPIを呼び出す：

```
https://us-central1-kikiki-flutter-template-prod.cloudfunctions.net/getMetricsSummary?period=monthly
```

## 出力形式

```markdown
## 📊 月次KPIレビュー - {今月}

### サマリー
| 指標 | 今月 | 前月比 |
|-----|-----|-------|
| 平均DAU | {数値} | {変化率}% |
| MAU | {数値} | - |
| 新規ユーザー | {数値} | - |
| セッション数 | {数値} | - |
| 推定収益 | ¥{数値} | - |

### アプリ別ランキング
| 順位 | アプリ名 | 平均DAU | MAU | 収益 |
|-----|---------|--------|-----|------|
| 1 | {アプリ名} | {DAU} | {MAU} | ¥{収益} |
| 2 | ... | ... | ... | ... |

### 月次分析
- 目標達成状況
- 成長トレンド
- 次月アクションアイテム
```
