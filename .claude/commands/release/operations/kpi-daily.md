# 日次KPIレビュー

全アプリの日次KPIを取得してレポートを作成してください。

## データ取得

`WebFetch` で以下のAPIを呼び出す：

```
https://us-central1-kikiki-flutter-template-prod.cloudfunctions.net/getMetricsSummary?period=daily
```

## 出力形式

```markdown
## 📊 日次KPIレビュー - {今日の日付}

### サマリー
| 指標 | 値 |
|-----|-----|
| 総DAU | {数値} |
| 新規ユーザー | {数値} |
| セッション数 | {数値} |
| 推定収益 | ¥{数値} |
| 対象アプリ数 | {数値} |

### アプリ別ランキング
| 順位 | アプリ名 | DAU | セッション | 収益 |
|-----|---------|-----|----------|------|
| 1 | {アプリ名} | {DAU} | {セッション} | ¥{収益} |
| 2 | ... | ... | ... | ... |

### 注目ポイント
- DAU上位アプリの特徴
- DAU 0のアプリがあれば警告
- 前日との比較（可能であれば）
```
