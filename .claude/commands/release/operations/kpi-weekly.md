# 週次KPIレビュー

全アプリの週次KPIを取得してレポートを作成してください。

## データ取得

`WebFetch` で以下のAPIを呼び出す：

```
https://us-central1-kikiki-flutter-template-prod.cloudfunctions.net/getMetricsSummary?period=weekly
```

## 出力形式

```markdown
## 📊 週次KPIレビュー - {今日の日付}

### サマリー
| 指標 | 今週 | 前週比 |
|-----|-----|-------|
| 平均DAU | {数値} | {変化率}% |
| 新規ユーザー | {数値} | - |
| セッション数 | {数値} | - |
| 推定収益 | ¥{数値} | - |

### アプリ別ランキング
| 順位 | アプリ名 | 平均DAU | セッション | 収益 |
|-----|---------|--------|----------|------|
| 1 | {アプリ名} | {DAU} | {セッション} | ¥{収益} |
| 2 | ... | ... | ... | ... |

### 週次分析
- 成長アプリ
- 減少アプリ
- 改善提案
```
