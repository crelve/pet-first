# 🎉 プロモーション戦略コマンド

<!-- LANGUAGE: ja -->
<!-- RESPONSE_LANGUAGE: Japanese -->

**ローンチ・イベント・キャンペーンのプロモーション戦略を自動生成**

## 🎯 概要

このコマンドは、以下のプロモーションシナリオに対応した戦略を生成します:
- 新規アプリローンチ
- メジャーアップデート
- 季節・イベント連動キャンペーン
- ユーザー獲得キャンペーン
- リエンゲージメント（休眠ユーザー復帰）

**IMPORTANT:** このコマンド実行前に、必ず以下を参照してください:
- **[トークン効率化ベストプラクティス](../../docs/best-practices/token-efficiency.md)** - 10個の黄金ルール

### 4. ユーザー獲得キャンペーン

```bash

   - 被紹介者: 特典（初月無料等）
   - 双方にメリット

実装:
   □ アプリ内「友達を招待」ボタン
   □ ユニークリンク生成
   □ SNS シェア機能（LINE, Twitter等）

   - 1人紹介 → 無料期間 +7日
   - 3人紹介 → 無料期間 +30日
   - 10人紹介 → 永久無料 or 限定バッジ

📊 目標:
   - バイラル係数（K-factor）: 0.3 以上
   - リファラル経由インストール: 全体の 15-25%
   - 紹介者の継続率: 通常ユーザーの 2倍

🏆 コンテスト・チャレンジ:

形式:
   - 期間限定（1-2週間）
   - 明確なルール・報酬
   - リーダーボード表示

例:
   「招待キャンペーン」
   - 期間: 2週間

   - ルール: 最も多く招待した人が勝ち

   「使用チャレンジ」
   - 期間: 1ヶ月
   - 報酬: 達成者全員に限定バッジ
   - ルール: 30日連続利用

💡 プロモーション:
   □ SNS でハッシュタグキャンペーン
   □ メール・プッシュ通知で告知
   □ アプリ内バナー掲載
   □ 進捗リマインダー通知」
```



```bash
「以下を含む包括的なプロモーション計画を生成:
```
