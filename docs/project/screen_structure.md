## 画面別仕様

### Splash
- アプリロゴ表示
- Hiveシード初期化（初回のみ）
- 1秒後 → Onboarding/Homeへ

### Onboarding
- 3ページのスワイプ式
- Page 1: コア価値
- Page 2: オフライン
- Page 3: カバー範囲
- "始める"ボタン → Home

### Home
- 上部: 犬/猫切り替えタブ
- 中央: カテゴリグリッド（2列）
  - 嘔吐・下痢 / 誤飲・中毒 / 痙攣 / 外傷 / 呼吸器 / 皮膚 / 眼 / 排尿
- 下部: バナー広告（フリーのみ）
- 右上: 設定アイコン

### SymptomList
- カテゴリ内の症状リスト
- 各項目: 症状名 + 緊急度プレビューバッジ
- タップ → SymptomDetail

### SymptomDetail
- 上部: 緊急度バッジ（大）+ 行動指示
- 中央: 「すぐにやること」ステップリスト
- 中段: 「やってはいけないこと」赤字警告
- 下段: 「病院に連れて行くサイン」チェックリスト
- 緊急度=赤のとき: "病院検索"ボタン（P1）
- 緊急度=緑のとき: バナー+下部広告

### Paywall
- ヒーローイメージ（犬・猫のイラスト）
- ヘッドライン: "ペットの命を守る、全機能解放"
- 3プラン横並び（年額にバッジ）
- 7日無料トライアル強調
- フッター: 利用規約・プライバシー

### Settings
- プレミアム購読状態
- プレミアムへアップグレード（フリーのみ）
- 通知設定
- プライバシーポリシー / 利用規約
- アプリ情報・バージョン
- レビューを書く

### PetProfile (P1)
- ペット一覧
- 追加ボタン
- 各ペット: 名前、種類、品種、年齢、体重、アレルギー、既往歴

### HospitalSearch (P1)
- 現在地ベース病院リスト
- 24時間対応フィルタ
- マップ表示
- 電話ボタン
