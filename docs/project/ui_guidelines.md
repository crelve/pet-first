# Pet First UIガイドライン

## デザイン原則

1. **緊急時に使えるUI**: 起動→症状タップ→ガイド表示が3タップ以内
2. **情報の優先順位を明確に**: 緊急度を最も大きく、応急処置ステップを次に
3. **NG行為は赤字で警告**: 見落とし防止
4. **オフライン明示**: 接続不要を強調

## コンポーネント

### EmergencyBadge
- 大きさ: 高さ48dp以上
- 色: 緊急度カラー
- アイコン+テキスト+行動指示
- 例: 🟥 すぐ病院へ / 🟨 様子見 / 🟩 自宅対処OK

### StepCard
- 番号付きカード
- ステップ番号は丸で強調
- 1ステップ1カード（情報の分離）

### NGWarningBox
- 赤背景
- 「✗」マーク
- "やってはいけないこと"の見出し

### PetTypeTab
- 犬/猫のアイコン+ラベル
- 選択中はプライマリカラー、未選択はグレー

## タイポグラフィ

| 種別 | サイズ | 重み |
|------|-------|------|
| Hero | 32sp | Bold |
| Heading 1 | 24sp | Bold |
| Heading 2 | 20sp | Bold |
| Body Large | 18sp | Regular |
| Body | 16sp | Regular |
| Caption | 14sp | Regular |
| Warning | 16sp | Bold（赤色） |

緊急時に読みやすく、本文は標準より気持ち大きめ。

## アニメーション

- ペイウォール表示: フェードイン+スケール
- 緊急度バッジ: 赤レベルは軽くパルス（注意喚起）
- 画面遷移: 標準material（過剰なエフェクトなし）

## アクセシビリティ

- Dynamic Type対応（iOS）
- フォントサイズ可変
- カラーだけに依存しない（緊急度は色+テキスト+アイコン）
- VoiceOver対応

## 多言語対応の注意

- 39言語対応
- 中国語・韓国語・アラビア語の文字レイアウト確認
- RTL（アラビア語・ヘブライ語）対応
- 医療用語は専門翻訳
