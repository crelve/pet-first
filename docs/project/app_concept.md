# Pet First アプリコンセプト

**アプリ名**: Pet First / Pet First – Emergency First Aid

## コンセプト

「うちの子が変。今すぐ確認、ペットの応急処置」

犬・猫の異常症状（嘔吐・下痢・痙攣・誤飲等）を選ぶだけで緊急度判定（赤:即病院/黄:様子見/緑:自宅対処OK）と正しい応急処置を即表示。深夜の「これ大丈夫？」を3秒で解決。

## エレベーターピッチ

ペット応急処置の専用アプリがほぼ空白。Red Cross Pet First Aidは情報過多で緊急時に使えない。Pet Firstは症状→緊急度色分け→応急処置の3ステップで「今すぐ病院か、朝まで待てるか」を即判定。first-aid（人間用）+pet-first（ペット用）で命を守るシリーズ。

## 差別化ポイント

1. **緊急度の色分け**: 赤/黄/緑で「今すぐ病院か」を3秒判定。情報過多にならない
2. **犬/猫別に最適化**: 種別ごとに異なる症状対応を分離。混乱しない
3. **オフライン完全対応**: 深夜・圏外でも100%動作。緊急時に使えるアプリ
4. **NG行為も明記**: やってはいけない処置を赤字で警告（誤飲時の催吐NG等）
5. **シリーズ展開**: first-aid（人間）+ pet-first（ペット）で家族全員カバー

## 訴求導線設計

- **ターゲットの痛み**: 深夜にペットが嘔吐→「病院連れてくべき？」とパニック。Google検索は情報散在で「獣医に聞いてください」で終わる
- **核心ベネフィット**: 3秒で「即病院か朝まで待てるか」が分かる。深夜の不安を即解消
- **フック案**:
  1. "うちの子が変。3秒で確認、ペット応急処置" — 深夜パニックに直撃
  2. "犬・猫の症状から、今すぐ病院か判定" — 機能を端的に
  3. "オフライン対応。深夜・圏外でも使える応急処置" — 信頼性訴求
- **反論処理**:
  - 「獣医に聞けばいい」→ 深夜は受付してない。3秒で初動判断ができる
  - 「ネット検索で十分」→ 情報散在で結論が出ない。緊急度色分けで即判定
  - 「他のアプリと違う？」→ Red Crossは情報過多。Pet Firstは緊急時UIに特化
- **CTA**: "今すぐインストール。深夜の不安、3秒で解消"
