## ペイウォール設計（M1必須）

### トライアル
- 7日間無料トライアル → 月額自動課金
- 初回起動時はペイウォール表示しない（即離脱防止）
- 3回目の症状検索後にペイウォール表示

### ペイウォール表示タイミング
1. オンボーディング後（ソフト訴求）
2. 3症状目検索時（ハード訴求）
3. ペットプロフィール作成時（P1）
4. 設定画面 > プレミアム

### 無料機能 / 有料機能

| 機能 | 無料 | プレミアム |
|------|------|-----------|
| 症状ガイド閲覧 | 1日3件まで | 無制限 |
| 緊急度判定 | 全症状OK | 全症状OK |
| 応急処置ステップ | 全症状OK | 全症状OK |
| 広告 | 表示 | 非表示 |
| ペットプロフィール（P1） | 1匹 | 無制限 |
| 病院検索（P1） | 3件まで | 無制限 |

## MVP範囲（M確定）

### P0（リリース必須）
- 犬/猫タブ切替
- カテゴリ別症状リスト（30+症状）
- 緊急度判定（赤/黄/緑）
- 応急処置ステップ表示
- NG行為警告
- オフライン完全対応
- ペイウォール（M1）
- バナー+インター広告（M2）
- ローカル通知（M3: 月1回の健康チェック促し）
- オンボーディング3ステップ（M4）
- D1/D7/D30 リテンション計測（M6）

### P1（リリース後1ヶ月以内）
- ペットプロフィール
- 動物病院検索（位置情報）
- 緊急電話ワンタップ

### P2（3ヶ月以内）
- ウサギ・ハムスター対応
- ワクチン記録
- 健康日記
