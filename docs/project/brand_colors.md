# ブランドカラー — Pet First

## コンセプト
**「うちの子が変。今すぐ確認、ペットの応急処置」**
深夜にペットが嘔吐・痙攣で飼い主がパニックになる場面を3秒で救うアプリ。
**安心感（Calm Sage Green）+ 緊急判定（Coral Alert Red）+ 信頼の自然色** で、医療系の冷たさではなく「家族の一員を守る温かさ」を表現する。

## 競合カラー分析

| アプリ | メインカラー | 印象 | 弱点 |
|--------|------------|------|------|
| Pet First Aid by Red Cross | Red Cross Red `#ED1B2E` | 救急感は強いが緊張・恐怖を煽る | UIが古い、深夜の不安をさらに増幅 |
| PetMD Symptom Checker | Medical Blue `#2D8DBF` | 医療っぽい清潔感 | 冷たい、家族感がない |
| ASPCA Animal Poison Control | Brand Orange `#F47521` | 警告感あり | 応急処置に特化していない |
| First Aid（人間用シリーズ） | Trust Blue `#3D8BFD` | 既存の姉妹アプリ | ペット版とは差別化必要 |

→ Red Cross の強い赤は緊急感が強すぎて深夜の不安を増幅する。
→ PetMD の医療ブルーは冷たく、ペットを家族として愛する温かさが出ない。
→ **Pet First は Calm Sage Green（落ち着き・癒し）をメインに、緊急アラート専用に Coral Red を限定使用** することで、深夜のパニックを鎮静化しつつ、本当に危険な症状は明確に警告する。

## 緊急度3色（このアプリの核）

| 緊急度 | カラー | 意味 |
|--------|-------|------|
| 🔴 赤 (Coral Alert `#FF6B6B`) | 即病院 | 痙攣・大量出血・誤飲・呼吸困難 |
| 🟡 黄 (Warm Amber `#FFB347`) | 様子見 | 軽い嘔吐・下痢・食欲低下 |
| 🟢 緑 (Calm Sage `#6FA572`) | 自宅対処OK | 軽度の傷・耳掃除・爪切り |

3色とも色弱対応のため明度差を確保。アイコン + テキスト + 色の3重伝達。

## 心理学的根拠

| 原則 | 適用 |
|------|------|
| Calm Sage Green | ヘルスケア・自然・回復のシグナル。深夜の不安を鎮める |
| Coral Red（彩度抑制） | 救急赤の強さは保ちつつ、鋭利な恐怖感を排除 |
| 暖色系（Coral / Amber） | ペットの体温・家族の温かさを連想 |
| 画面下部の Sage Green | 不安レベルを下げる「落ち着け」のシグナル |

## 差別化キーワード

- **温かい安心感**: Red Cross の恐怖を煽る赤と差別化
- **ファミリー医療感**: PetMD の冷たい医療色と差別化
- **明確な3段階判定**: 信号機方式で「今すぐ動くべきか」が一瞬で分かる
- **first-aid シリーズの親しみ**: 人間用 first-aid の姉妹アプリとして識別可能
