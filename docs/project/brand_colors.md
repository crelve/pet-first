# Pet First ブランドカラー

## カラーパレット

### プライマリ
| 色名 | 用途 | Hex | 説明 |
|------|------|-----|------|
| Primary | ブランド色（ボタン、AppBar） | `#2E7D6B` | 信頼の濃い緑、医療系の安心感 |
| Primary Light | アクセント | `#5BA897` | 柔らかい緑 |
| Primary Dark | 強調 | `#1B5945` | 深い森の緑 |

### 緊急度カラー（コア価値）
| レベル | Hex | 用途 |
|--------|-----|------|
| Emergency Red | `#D32F2F` | 即病院。背景・バッジ |
| Warning Yellow | `#FFA000` | 様子見・要観察 |
| Safe Green | `#388E3C` | 自宅対処OK |

### ニュートラル
| 色名 | Hex | 用途 |
|------|-----|------|
| Background | `#FAFAFA` | 画面背景 |
| Surface | `#FFFFFF` | カード |
| Text Primary | `#212121` | 主要テキスト |
| Text Secondary | `#757575` | サブテキスト |
| Divider | `#E0E0E0` | 区切り線 |

## カラーの選定理由

- **緑系をプライマリ**に: 医療系・信頼・癒し。Red CrossやPetMDの赤系と差別化
- **緊急度色は強烈に**: 一目で危険度がわかるサイン色。視認性最優先
- **コントラスト**: WCAG AA準拠（4.5:1以上）

| 色 | Light | Dark |

| Text Primary | `#212121` | `#FFFFFF` |

緊急度カラーはダーク/ライト共通（識別性優先）
