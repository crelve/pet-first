## 差別化戦略

### 強み（既存アプリと比較）

1. **緊急度色分け**: Red Crossは情報過多→赤/黄/緑で3秒判定
2. **犬猫別UI**: 種別タブで混乱防止
3. **NG行為の明記**: 誤飲時の催吐禁止等を赤字警告
4. **オフライン100%**: PetMDはWebサイト型→電波なしで使えない
5. **シリーズ展開**: first-aid（人間用）と並べて訴求

### 弱み（受け入れる）

- 医療監修なし（Red Cross優位） → 信頼できる獣医情報源を引用で代替
- ペット記録機能なし → P1で追加検討、まずは応急処置に特化

## 結論

ペット応急処置の専用アプリは空白。Red Crossは医療監修が強みだがUIが古く緊急時に使いづらい。Pet Firstは「3秒で緊急度判定」の使いやすさで差別化。first-aidシリーズとしてのブランド統一でCPI削減効果。
