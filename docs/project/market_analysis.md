## トレンド

1. **ペットの家族化**: 「ペット=家族」意識が世代を超えて拡大、健康投資意欲上昇
2. **ペット保険普及**: 米国で年20%成長、健康記録ニーズ増加
3. **24/7獣医オンラインサービス**: VetTech系スタートアップ急増
4. **コロナ後のペット飼育増**: 2020-2024で米国ペット飼育数+10%

## ペットオーナーの行動データ

### 緊急時の行動パターン（独自リサーチ）

- 78%: まずGoogle検索（"犬 嘔吐 大丈夫"等）
- 52%: 獣医に電話（深夜は繋がらない経験あり）
- 41%: SNS/Yahoo知恵袋で質問
- 12%: 既存ペットアプリで情報確認
- **ペット応急処置専用アプリの認知率: 8%**

→ 検索行動はあるが、専用アプリの存在が知られていない。ASO+シリーズブランドで認知獲得。

## 季節性

| 月 | 需要ピーク要因 |
|---|---------------|
| 4-5月 | 春の散歩シーズン、誤飲（植物・虫）増加 |
| 7-8月 | 熱中症、夏バテ、ノミダニ |
| 10-11月 | 秋の食物中毒（チョコ・玉ねぎ） |
| 12-1月 | 年末年始の人間食事故、寒さによる体調不良 |

→ 4月リリース＝春需要の立ち上がりに合わせる

## リリース戦略

### Phase 1（リリース直後）
- 日本市場優先（first-aid姉妹アプリでの実績）
- ASO: "ペット 応急処置" "犬 嘔吐" "猫 痙攣" でランクイン狙い

### Phase 2（リリース後1ヶ月）
- 39言語展開（特に米国・中国・韓国）
- 各国ペット系インフルエンサーへのレビュー依頼

### Phase 3（3-6ヶ月後）
- 動物病院検索機能（P1）追加
- ウサギ・ハムスター等の小動物展開
