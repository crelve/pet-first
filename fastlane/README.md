# Fastlane Directory

This directory contains App Store Connect upload preparation artifacts.

## Directory Structure

- `screenshots_upload/` - Temporary directory for App Store Connect upload
  - Generated by `prepare_screenshots_for_upload` lane
  - Contains best version of each screenshot (AI > framed > original)
  - **Automatically cleaned and regenerated** - do not edit manually

## Official Fastfile Location

The main Fastfile is located at: **`ios/fastlane/Fastfile`**

All fastlane commands should be run from `ios/` directory:

```bash
cd ios
bundle exec fastlane <lane_name>
```

## Screenshot Storage

Screenshots are stored at project root:
- **`screenshots/`** - Official screenshot directory (source of truth)
  - `screenshots/ja/`, `screenshots/en-US/`, etc.
  - Original: `01-home.png`
  - Framed: `01-home_framed.png`
  - AI-generated: `01-home_ai.png`
